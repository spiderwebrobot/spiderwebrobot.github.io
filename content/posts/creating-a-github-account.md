You’ll need an email address to [sign up](https://github.com/signup) for GitHub. [Gmail](https://support.google.com/mail/answer/56256) is a solid choice if you don’t already have one. A Gmail account allows you to [Sign in with Google](https://www.google.com/account/about/sign-in-with-google/), which is a key that can unlock many doors. That being said, if you’d rather not put all of your eggs in one basket, here are some alternatives:

- [Microsoft Outlook](https://www.microsoft.com/en-us/microsoft-365/outlook/email-and-calendar-software-microsoft-outlook)
- [Proton Mail](https://proton.me/mail)
- [Yahoo Mail](https://mail.yahoo.com/)

## You’ve got mail

Once you have an email you can start [Creating an account on GitHub](https://docs.github.com/en/get-started/start-your-journey/creating-an-account-on-github). A GitHub account also unlocks many doors including [Cloudflare](https://www.cloudflare.com/) and [CodePen](https://codepen.io/) to name a few.

## A word on passwords

The world is your oyster. So where to now? If you are only managing your email and GitHub accounts, then you can probably get away with remembering your passwords.

However, if you find yourself managing several passwords, consider using a password manager to keep your accounts secure. If you can afford it [1password](https://1password.com/pricing#personal) is highly ranked, otherwise, [Bitwarden](https://bitwarden.com/pricing/) will get you started for free, or [NordPass](https://nordpass.com/plans/) offers a solution that can be bundled with NordVPN.
